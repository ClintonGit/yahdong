# Backend/Boards Requirements — v1 (2026-04-30)

## Scope
Kanban board config — default column seed, column reorder

## Requirements

### REQ-001: Seed Default Statuses
- [ ] เมื่อ POST /projects สร้างสำเร็จ → seed 3 statuses อัตโนมัติ
- [ ] Default columns: `Backlog` (order:1), `In Progress` (order:2), `Done` (order:3)
- [ ] Default colors: `#94A3B8`, `#F59E0B`, `#22C55E`

### REQ-002: Reorder Columns
- [ ] PATCH /projects/:id/statuses/reorder
- [ ] Body: `{ orderedIds: string[] }` — array of statusId ตามลำดับใหม่
- [ ] Requires: ProjectGuard role != 'viewer'
- [ ] Update order field ของทุก status ใน transaction

## Notes
- Seed ทำใน Projects service หลัง create project สำเร็จ
