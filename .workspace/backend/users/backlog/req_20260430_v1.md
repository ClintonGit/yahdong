# Backend/Users Requirements — v1 (2026-04-30)

## Scope
User profile — ดู/แก้ไข profile ตัวเอง, ดู public profile คนอื่น

## Requirements

### REQ-001: Get My Profile
- [ ] GET /users/me
- [ ] Requires: JwtAuthGuard
- [ ] Return: `{ id, name, email, avatar, createdAt }`

### REQ-002: Update My Profile
- [ ] PATCH /users/me
- [ ] Requires: JwtAuthGuard
- [ ] Body: `{ name?, avatar? }` (optional fields)
- [ ] Validate: name min 2 chars, avatar ต้องเป็น URL
- [ ] Return: updated user object

### REQ-003: Get Public Profile
- [ ] GET /users/:id
- [ ] Requires: JwtAuthGuard
- [ ] Return: `{ id, name, avatar }` เท่านั้น (ห้าม expose email)

## Notes
- avatar เป็น URL string (Phase 1) — upload จริงทำ Phase 2
